```bash

```

```bash
uuidgen | tr [:upper:] [:lower:]

# ...some uuid
```

> **Warning**: You need quotes (`'`) if using `zsh`: https://stackoverflow.com/questions/62266314/zsh-no-matches-found-space
>
> ```bash
> uuidgen | tr '[:upper:]' '[:lower:]'
> ```
