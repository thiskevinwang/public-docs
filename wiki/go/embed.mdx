# How to embed static files with a `go` binary

Import `embed`. The underscore will prevent potential
autoformatting from removing the immport.

Then use the `go:embed` directive.

```go
import (
	_ "embed"
)

//go:embed templates/wiki-template.mdx.tmpl
var templateFileContents string
```
