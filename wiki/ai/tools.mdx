### Mods

https://github.com/charmbracelet/mods
https://github.com/charmbracelet/glow

```
export OPENAI_API_KEY=...
```

```bash
# optional
ISO_DATE=$(date -u +"%Y-%m-%dT%H:%M:%SZ")
# the DATE function is also available
```

```bash
mods -f "what day of the week is $(DATE)?" | glow
# The day of the week for Sat Jun 24 10:03:59 EDT 2023 is Saturday.

mods -f "what week of the year does $(DATE) fall under?" | glow
# The week of June 24, 2023 falls under Week 25 of the year 2023.
```

```bash
# ~/.zshrc
function ask() {
  mods -f "$1" | glow
}
```
